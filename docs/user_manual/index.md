# Руководство пользователя Finam Terminal

**Finam Terminal** — это терминальное приложение для торговли на фондовом рынке через брокера Финам. Программа работает прямо в консоли и позволяет отслеживать портфель, просматривать котировки, анализировать графики и совершать торговые операции — всё без необходимости открывать браузер или тяжёлый торговый терминал.

## Возможности

- Просмотр портфеля и позиций по нескольким счетам
- История сделок за последние 30 дней
- Управление активными заявками
- Поиск инструментов с котировками в реальном времени
- Профиль инструмента со свечным графиком
- Создание заявок: рыночные, лимитные, стоп-лосс, тейк-профит, связанные SL+TP
- Автоматическое обновление данных
